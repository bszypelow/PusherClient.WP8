# Pusher WP8 Client library

This is a Windows Phone 8 library for interacting with the Pusher WebSocket API.

Registering at <http://pusher.com> and use the application credentials within your app as shown below.

More general documentation can be found at <http://pusher.com/docs/>.

## Installation

### NuGet Package
```
Install-Package PusherClient.WP8 -Pre
```
